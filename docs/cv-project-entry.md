# Entrée CV - Développement IA & DevOps

## Version courte

**Assistant IA documentaire RAG - Développement IA & DevOps**

- Conception et développement d'un assistant IA basé sur une architecture RAG pour interroger des documents indexés.
- Mise en place du pipeline IA : extraction multi-format, OCR, nettoyage du texte, chunking, embeddings et recherche sémantique avec Qdrant.
- Intégration de LiteLLM et de modèles Gemini pour centraliser les appels LLM, générer les embeddings et produire des réponses avec sources.
- Développement d'une API FastAPI et d'une interface React/TypeScript pour l'import de documents, le chat et l'affichage des sources utilisées.
- Déploiement local avec Docker Compose : API, LiteLLM Proxy, Qdrant et PostgreSQL, avec configuration par variables d'environnement.

**Stack :** Python, FastAPI, React, TypeScript, LiteLLM, Gemini, Qdrant, PostgreSQL, Docker, Docker Compose.

## Version CV avec axes IA et DevOps

**Projet personnel - Assistant IA documentaire RAG**  
**Rôle : Développeur IA & DevOps**

**Développement IA**

- Développement d'un système RAG permettant d'indexer des documents et de répondre aux questions à partir d'un contexte documentaire.
- Implémentation du pipeline d'extraction : PDF, images avec OCR, fichiers Office, fichiers texte et données structurées.
- Génération d'embeddings via LiteLLM, stockage vectoriel dans Qdrant et recherche sémantique des passages pertinents.
- Construction des prompts système pour contrôler l'utilisation des sources et améliorer la traçabilité des réponses.

**DevOps**

- Conteneurisation de l'application avec Docker et orchestration des services avec Docker Compose.
- Configuration des services API FastAPI, LiteLLM Proxy, Qdrant et PostgreSQL dans un environnement local reproductible.
- Gestion des variables d'environnement pour les clés API, modèles LLM, dimensions d'embeddings et connexions aux services.
- Préparation d'une architecture modulaire facilitant le déploiement, la maintenance et l'évolution du projet.

## Version portfolio ou LinkedIn

Conception et développement d'une application RAG full-stack permettant de poser des questions sur des documents indexés. Le projet combine une API FastAPI, un pipeline IA d'extraction, d'OCR, de chunking et d'embeddings, une base vectorielle Qdrant, un proxy LiteLLM et une interface React/TypeScript. La partie DevOps repose sur Docker Compose pour orchestrer l'API, LiteLLM, PostgreSQL et Qdrant dans un environnement local reproductible.
